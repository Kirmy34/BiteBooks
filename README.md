# BiteBooks

BiteBooks is your personal digital cookbook. It's a beginner-friendly full-stack web application designed for adding, viewing, searching, and organizing your favorite recipes.

This project is a great introduction to full-stack development, using a Django backend and a Vue.js frontend.

## Getting Started

The best way to develop on BiteBooks is using **Docker** and **VS Code Dev Containers**.

### Prerequisites

*   [Docker](https://www.docker.com/get-started) and [Docker Compose](https://docs.docker.com/compose/install/)
*   [VS Code](https://code.visualstudio.com/) with the [Dev Containers extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)

### Using Dev Containers

1.  Open the project folder in VS Code.
2.  When prompted, click **"Reopen in Container"**.
3.  The environment will be automatically set up with all dependencies and a PostgreSQL database.

## Tech Stack

*   **Frontend:** Vue 3 (Composition API), TypeScript, Tailwind CSS 4.x, Vite.
*   **Backend:** Django 6.0.2, Django REST Framework, Simple JWT.
*   **Database:** PostgreSQL 16.

Happy cooking (and coding)!
